Implementations of the numerical experiments are in Pluto.jl notebooks.

To run them launch julia

If Pluto is not installed: 
```julia-repl
julia> using Pkg
julia> Pkg.add("Pluto")
```

Run Pluto:
```julia-repl
julia> using Pluto
julia> Pluto.run()
```

Then the notebooks should be accessible from the Pluto web-server.
